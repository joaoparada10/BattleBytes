Animates a sprite. 

Animate the sprite "{}fig"{} according to its attributes in memory, whose address is "{}base"{}. The animation detects the screen borders and change the speed according; it also detects collision with other objects, including the player pad. Collisions with the screen border generates a perfect reflection, while collision with other objects generates a random perturbation in the object speed. Other strategies can be devised\+: use quasi-\/elastic collision based on the objects "{}mass"{} and speed, generate spin effect based on amount of tangential speed direction relative to the object center of "{}mass"{}... Returns the kind of collision detected, RIGHT\+\_\+\+HIT or LEFT\+\_\+\+HIT, if a collision with the players pad (WHITE colored!) is detected.

This function updates the sprite\textquotesingle{}s position based on its speed and then draws the sprite at its new position.

Creates a sprite. 

Creates with random speeds (not zero) and position (within the screen limits), a new sprite with pixmap "{}pic"{}, in memory whose address is "{}base"{}; Returns NULL on invalid pixmap.

This function allocates memory for a sprite object and initializes it with the provided pixmap, position, and speed.
