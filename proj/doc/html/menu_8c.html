Creates a new menu.

Parameters
spPointer to the sprite representing the menu background.
titleThe title of the menu.
title_xX coordinate for the title.
title_yY coordinate for the title.
optionsArray of strings representing the menu options.
options_xArray of X coordinates for each option.
options_yArray of Y coordinates for each option.
options_x_hitboxArray of X coordinates for the hitboxes of each option.
options_y_hitboxArray of Y coordinates for the hitboxes of each option.
options_heightArray of heights for each option's hitbox.
options_widthArray of widths for each option's hitbox.
options_stateArray of states associated with each option.
num_optionsThe number of options in the menu.
Returns
Pointer to the newly created menu or NULL if memory allocation fails.
