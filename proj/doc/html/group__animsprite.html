int animate_asprite ( AnimSprite * asp,
bool destroy )

Animates an animated sprite.

Animate an Animated Sprite

This function updates the animated sprite's current frame according to its animation speed. If the animation speed allows for it, the sprite's pixmap is updated to the next frame.

Parameters
aspThe animated sprite to animate
destroyFlag indicating whether the sprite should be destroyed after reaching the last frame
Returns
1 if the last frame is reached and destroy flag is set, 0 otherwise

Creates an animated sprite.

Create an Animated Sprite from multiple pixmaps At least one pixmap must be specified.

This function creates an animated sprite with the provided parameters. It initializes the sprite with the first pixmap and allocates memory for an array of pointers to store subsequent pixmaps.

Parameters
aspeedThe animation speed
no_picThe number of pixmap frames
pic1The first pixmap frame
...Additional pixmap frames (variadic arguments)
Returns
A pointer to the created animated sprite, or NULL if creation fails
