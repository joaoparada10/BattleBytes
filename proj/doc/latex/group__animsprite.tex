Animates an animated sprite. 

Animate an Animated \doxylink{struct_sprite}{Sprite}

This function updates the animated sprite\textquotesingle{}s current frame according to its animation speed. If the animation speed allows for it, the sprite\textquotesingle{}s pixmap is updated to the next frame.

Creates an animated sprite. 

Create an Animated \doxylink{struct_sprite}{Sprite} from multiple pixmaps At least one pixmap must be specified.

This function creates an animated sprite with the provided parameters. It initializes the sprite with the first pixmap and allocates memory for an array of pointers to store subsequent pixmaps.
